# Domain Model Specification (Stage 2) - Part 1: Domain Overview

## 1. DOMAIN OVERVIEW

TalePick เป็นแพลตฟอร์มนิทานสนทนาภาษาไทยที่ผู้ใช้นำทางเรื่องราวแบบแยกเส้นทางผ่านฉากที่เสริมด้วยมัลติมีเดีย โดยการตัดสินใจเลือกที่มีกลยุทธ์ซึ่งใช้เครดิตและปลดล็อกจุดสิ้นสุดของเรื่องที่แตกต่างกัน แพลตฟอร์มนี้ใช้เศรษฐกิจที่ใช้เครดิตเป็นหลัก ซึ่งผู้เล่นต้องจัดการทรัพยากรเพื่อสำรวจเส้นทางของเรื่อง โดยความสำเร็จทำหน้าที่เป็นทั้งตัวบ่งชี้ความคืบหน้าและกุญแจสำหรับเนื้อหาพิเศษ

ปัญหาหลักที่แก้ไขคือการสร้างประสบการณ์นิทานสนทนาที่น่าสนใจพร้อมกลไกการจัดการทรัพยากรที่ส่งเสริมการรักษาผู้เล่นและการสร้างรายได้ ผู้เล่นก้าวหน้าในเรื่องราวโดยการตัดสินใจที่มีความหมายซึ่งส่งผลต่อผลลัพธ์ของเรื่อง ในขณะที่ระบบเครดิตสร้างการตัดสินใจเชิงกลยุทธ์เกี่ยวกับเส้นทางที่จะสำรวจ

แนวคิดโดเมนระดับสูงประกอบด้วย: ผู้ใช้ (ผู้เยี่ยมชมและผู้สมัครสมาชิก) ที่บริโภคเนื้อหา; เรื่องราวที่มีฉากแยกเส้นทางเชื่อมต่อด้วยตัวเลือก; เศรษฐกิจเครดิตที่มีการเติมเต็มตามเวลาและตัวเลือกการซื้อ; ความสำเร็จที่ปลดล็อกทั้งการปรับแต่งอวาตาร์และเนื้อหาเรื่อง; และการจัดอันดับที่ให้ข้อเสนอแนะและรางวัลเครดิตเพิ่มเติม

ปฏิสัมพันธ์ระหว่างองค์ประกอบโดเมนสร้างระบบนิเวศที่ซับซ้อน: ผู้เล่นใช้เครดิตเพื่อตัดสินใจเลือกที่นำทางระหว่างฉาก นำไปสู่จุดสิ้นสุดของเรื่องที่แตกต่างกัน ความสำเร็จที่ได้จากการเสร็จสิ้นเรื่องจะปลดล็อกตัวเลือกพิเศษและการปรับแต่งอวาตาร์ การจัดอันดับให้ข้อเสนอแนะแก่ผู้เขียนและรางวัลผู้เล่นด้วยเครดิต ผู้ดูแลระบบดูแลระบบนิเวศทั้งหมดผ่านเครื่องมือวิเคราะห์และการจัดการเนื้อหาขั้นสูง
